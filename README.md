# Neo4j-Professional Social Media Database
 Built a LinkedIn-style professional social network using Neo4j for my college analytics assignment in semester 5. 
 
# What I Built

- **Nodes**:
  - Users
  - Companies
  - Skills
  - Jobs]

- **Relationships**:
  - Users *have skills* 
  - Users *work at* 
  - Users *connected to* other users 
  - Companies *post* job openings
  - Users *apply to* jobs

# Example Queries
- Find all connections of a user
- Suggest friends-of-friends connections
- List all users with a specific skill
- Find jobs matching a user's skills
- List all employees of a company
